## 📋 Quick Setup (5 Minutes)

### Step 1: Download Requirements
1. **Download Node.js**: Go to [nodejs.org](https://nodejs.org) → Download LTS version → Install
2. **Download your code**: Click "Download Code" button above → Extract ZIP file

### Step 2: Run Your Website
1. **Open Terminal/Command Prompt**:
   - Windows: Press `Win + R` → type `cmd` → Enter
   - Mac: Press `Cmd + Space` → type `terminal` → Enter

2. **Navigate to your code folder**:
   \`\`\`bash
   cd path/to/your/extracted/folder
   \`\`\`

3. **Install and run**:
   \`\`\`bash
   npm install
   npm run dev
   \`\`\`

4. **Open your browser**: Go to `http://localhost:3000`
